1. Install Angular
	
2. Install angular material
	-> npm install --save @angular/material@6.4.7
	
	-> We need cdk for angular material
		npm install --save @angular/cdk@6.4.7

	-> Install angular animation
		npm install --save @angular/animations@6.1.7

	-> Install HammerJS: used to supports gestures in Angular
		npm install hammerjs@2.0.8 --save

3. Install Angular-Flexbox
	-> npm install --save @angular/flex-layout@6.0.0-beta.18

4. Install Font-Awesome:
	-> npm install font-awesome@4.7.0 --save

5. Install JSON server:
	-> npm install -g json-server
